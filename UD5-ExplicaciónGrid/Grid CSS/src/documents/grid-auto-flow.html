La propiedad Grid-auto-flow

La propiedad CSS grid-auto-flow forma parte de la especificación CSS Grid Layout que nos permite controlar el flujo de los elementos de la cuadrícula que se colocan automáticamente cuando no se posicionan explícitamente con ninguna propiedad grid de posicionamiento.

Nota! Establecemos la propiedad grid-auto-flow en dense para que el algoritmo de auto-colocación haga el back-tracking para encontrar y rellenar huecos con elementos más pequeños que vienen después en el grib. En definitiva, autoajusta el contenido para que no queden huecos.

Fíjate en el noveno elemento de la cuadrícula, ahora se ha movido para rellenar el primer hueco en el que puede caber, de la misma forma que el séptimo. Así que, por defecto, la cuadrícula avanza y nunca mira hacia atrás para comprobar si puede encajar un elemento en los espacios vacíos anteriores. Pero cuando declaramos denso, el algoritmo intenta rellenar tantos huecos como pueda, independientemente del orden de origen.

Fila denso

Como row es el valor por defecto, usar row dense es lo mismo que usar dense y tiene el mismo efecto.

columna densa

Los elementos de la rejilla se ordenarán por columnas mientras se rellenan los huecos.
