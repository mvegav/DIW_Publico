Ejemplo de ventanas modal básicas

Ventana modal básica

En este caso metemos en un fichero js el código fuente y capturamos el evento.

Codigo html

<button id="open">Ventana modal<button>
  <dialog id="modal">
  <h2>Este es el título de mi ventana modal</h2>
  <p>Este es un texto de ejemplo dentro de una ventana modal</p>
  <button id="close">Ventana modal<button>
</dialog>

Ventana Modal normal

..show() - al hacer clic en el elemento que lleve este atributo, abrirá el elemento dialog.

.close() - al hacer clic en el elemento que lleve este atributo, cerrará el elemento dialog.

Como podemos ver mientras no cerremos esta ventana no voleveremos a la anterior.

Empleamos el pseudo-elemento ::backdrop podemos cambiar la tonalidad y buscar un color que se ajuste más a nuestro diseño.


          dialog::backdrop {
            background: rgba(234,206,227,0.9)
          }

Ventana Modal ShowModal

.showModal() - al hacer clic en el elemento que lleve este atributo, abrirá el elemento dialog, bloqueando la interacción con el resto de elementos.

.close() - al hacer clic en el elemento que lleve este atributo, cerrará el elemento dialog.

Como podemos ver mientras no cerremos esta ventana no voleveremos a la anterior.

Empleamos el pseudo-elemento ::backdrop podemos cambiar la tonalidad y buscar un color que se ajuste más a nuestro diseño.


            dialog::backdrop {
             background: rgba(234,206,227,0.9)
            }
