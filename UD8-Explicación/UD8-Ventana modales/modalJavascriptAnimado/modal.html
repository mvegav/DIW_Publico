Ejemplo de ventanas modal animada

Ventana modal con animación

En este caso emplemos animaciones y javascript para mostrar y cerrar los diálogos.

Datos HTML- * Atributo

El data-*atributo se usa para almacenar datos personalizados privados para la página o la aplicación.

El data-*atributo nos brinda la capacidad de incrustar atributos de datos personalizados en todos los elementos HTML.

Los datos almacenados (personalizados) se pueden usar en el JavaScript de la página para crear una experiencia de usuario más atractiva (sin llamadas Ajax ni consultas a la base de datos del lado del servidor).

Para poder usarlo en javascript emplearemos dataset.

Podemos emplear este atributo html para añadir información al elemento que posteriormente emplearemos en el javascript, pero en nuestro caso lo emplearemos para indicar la animación a aplicar al elemento. De esta forma podremos reutilizar las animaciones.

Para usarlo solo debemos poner el nombre al atributo empezando por data, seguido por un guión medio y un nombre en minúsculas.

Ejemplo


          <article id="electric-cars"
          data-columns="3"
          data-index-number="12314"
          data-parent="cars" article>

Para poder obtener los datos indicados en data emleamos dataset en javascript.


        const articulo = document.querySelector("#electric-cars");
        articulo.dataset.columns; // "3"
        articulo.dataset.indexNumber; // "12314"
        articulo.dataset.parent; // "cars"

En el caso del css

Por ejemplo, para mostrar los datos principales en el artículo, puede usar contenido generado en CSS con la attr()función: 


        articulo::before {
          content: attr(data-parent);
        }

O puedes usar los selectores de atributos en CSS para cambiar estilos según los datos:


        articulo[data-columns="3"] {
          width: 400px;
        }
        articulo[data-columns="4"] {
          width: 600px;
        }
