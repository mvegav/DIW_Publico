Ejemplo de ventanas modal básicas

Ventana modal básica

A la hora de crear ventanas modales básicas podemos:

  • Emplear el elemento dialog de html y javascript

    El elemento dialog representa una parte del documento pensada para la interacción con el usuario, como por ejemplo, un cuadro de diálogo, un inspector o una ventana.

    Empleando el elemento html dialog y javascript para abrir la venta modal.

  • Empleando solo html y css

    En este caso lo que empleamos enlaces a partes de una página web y la pseudo-clase target, la cual representa a un elemento único representado por un id en el html.

    Ventana modal solo CSS

El elemento dialog

El elemento dialog es un elemento html compatible con todos los navegadores actuales menos con Internet Explorer, KaiOS Browser y Opera Mini.

Podemos verlo en caniuse

¿Cómo funciona el elemento Dialog?

Por defecto todo lo que pongamos en un elemento dialog está oculto. De forma que si ponemos el siguiente código no se verá nada en la pantalla. 


        <dialog>
        <h2>Este es el título de mi ventana modal</h2>
        <p>Este es un texto de ejemplo dentro de una ventana modal</p>
        </dialog>

Para mostrar el contenido de dentro de un elemento dialog, solo hay que añadir el atributo open, quedando algo así: 


        <dialog open>
        <h2>Este es el título de mi ventana modal</h2>
        <p>Este es un texto de ejemplo dentro de una ventana modal</p>
        </dialog>

Creación de la ventana de diálogo

Para crear la ventana modal necesitamos:

  • El elemento dialog que contendrá la ventana modal. Le pondremos un id por si tenemos más de una.
  • Un elemento de apertura de la ventana modal, enlace o botón. Emplearemos javascript para abrirla.
  • Un elemento de cierre de la ventana modal, enlace o botón. Emplearemos javascript para abrirla.

          <button onclick="windows.modal.showModal();">Ventana modal<button>
          <dialog id="modal">
          <h2>Este es el título de mi ventana modal</h2>
          <p>Este es un texto de ejemplo dentro de una ventana modal</p>
          <button onclick="windows.modal.close();">Ventana modal<button>
          </dialog>

Tipos de ventanas diálogo

Existen dos tipos de ventanas de diálogo: las modales (ventana de diálogo en la que el usuario no puede interactuar con otros elementos de la página) y no modales.

Para crear las ventanas de diálogo lo único que tenemos que hacer es usar el método show (para las ventanas no normales) o showModal (para las ventanas de diálogo modales).

Ventanas de diálogo no modales

Las ventanas no modales se utilizan si la información solicitada no es obligatoria para continuar, por lo que la ventana se puede dejar abierta mientras se continúa el trabajo. Por ejemplo, una barra de herramientas es un cuadro de diálogo no modal, y sus elementos se pueden utilizar para elegir características particulares de la aplicación. Normalmente, un diseño de software ideal invocaría este tipo de cuadros de diálogo tanto como sea posible. No obligan al usuario a realizar ningún modo específico de funcionamiento... lo que significa que pueden trabajar a su propio ritmo, en el orden que más les convenga.

Ventas de diálogo modales

Las ventanas modales o modal box, son in recuadro que aparece sobre la página, con la particularidad de que bloquea todas las funciones y concentra el foco en una acción particular, de modo que el usuario solamente puede hacer esa acción o cerrar la ventana.

Ventana Modal normal

show() - al hacer clic en el elemento que lleve este atributo, abrirá el elemento dialog.

close() - al hacer clic en el elemento que lleve este atributo, cerrará el elemento dialog.

Como podemos ver para cerrar esta ventana modal debemos pulsar el botón de cerrar pero el usuario tendrá interacción con el resto de la página pudiendo hacer click en otras partes de la misma.

Ventana Modal ShowModal

.showModal() - al hacer clic en el elemento que lleve este atributo, abrirá el elemento dialog, bloqueando la interacción con el resto de elementos.

.close() - al hacer clic en el elemento que lleve este atributo, cerrará el elemento dialog.

Como podemos ver mientras no cerremos esta ventana no voleveremos a la anterior.

Empleamos el pseudo-elemento ::backdrop podemos cambiar la tonalidad y buscar un color que se ajuste más a nuestro diseño.


            dialog::backdrop {
             background: rgba(234,206,227,0.9)
            }
