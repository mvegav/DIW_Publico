Ventana de diálogo básica

Las ventanas de diálogo se encuentran entre los elementos de diseño más efectivos que un diseñador web puede emplear.

Este tipo de elemento se diferencia de los típicos pop ups o ventanas emergentes en que no aparecen de forma automática, sino que es el propio usuario el que tiene que clicar en alguna parte de la web (normalmente un botón) para que aparezca.

Antes de continuar, aclaremos un poco de terminología potencialmente confusa en el mundo del diseño de interfaces de usuario: la diferencia entre elementos de diseño modales y no modales (también conocidos como no modales).

Los cuadros (ventanas) de diálogo modales no permitirán a los usuarios trabajar con la ventana principal una vez que se invocan; mientras que los cuadros de diálogo no modales permitirán a los usuarios acceder a la ventana principal, incluso después de haberlos invocado.

Disponemos de diversas formas de crear ventanas de diálogo:

  • Empleando solo html y CSS con la pseudo-clase target.

    Ventana modal solo CSS

  • Empleando el elemento html dialog y javascript para abrir la venta modal.

    Ejemplo

    Ventana modal con dialog

A la hora de realizar ventanas modales con javascript podemos:

  • Usar el javascript directamente en el html, con lo cual ese código no podemos reutilizarlo.
  • Usar el javascript en un documento externo. Esta opción nos permite separar la parte visual de la parte de código y reutilizar ese script en diversas páginas.

También podemos meter dentro de la ventana modal un formulario y escribir el resultado en la ventana principal.
